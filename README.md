# ניהול עבודה חקלאית — רגבים בנימין

ריפו זה מכיל את **אפליקציית ניהול העבודה החקלאית** (תיקיית `ניהול-חקלאות`).

## הפעלה
פותחים את `ניהול-חקלאות/index.html` בדפדפן (או מריצים את `ניהול-חקלאות/הפעלה.bat`).
הוראות מלאות: `ניהול-חקלאות/הוראות שימוש.md`.

## נתונים
- `ניהול-חקלאות/data.json` — נתוני האפליקציה
- `ניהול-חקלאות/ניהול- חקלאות.json` — ייצוא/גיבוי נתונים

## מה עוד היה כאן בעבר?
- **אפליקציית התקציב** — הגרסה החיה נמצאת בריפו [regavim-budget](https://github.com/guytsuberi1/regavim-budget); עותק ישן שהיה כאן הוסר (שמור בהיסטוריית git).
- **מסמכי משרד** (אישורי קרקע, דיווחי תקנה) — שמורים בהיסטוריית git ובתיקיית הגיבוי בגוגל דרייב.
