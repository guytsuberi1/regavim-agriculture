# ניהול עבודה חקלאית — רגבים בנימין

אפליקציה לניהול העבודה החקלאית: סידור יומי, תכנון שבועי, דרישת תשלום לגבייה ודוחות.
כל הנתונים נשמרים אצלכם (ב-OneDrive) — ללא חשבונות, ללא אינטרנט חיצוני.

## איך מפעילים

### דרך מומלצת — שמירה אוטומטית מלאה ל-OneDrive (ללא לחיצות)
1. לחצו פעמיים על **`הפעלה.bat`**.
2. ייפתח חלון שחור (השרת) ואז הדפדפן עם האפליקציה. אפשר למזער את החלון השחור.
3. זהו — **כל שינוי נשמר אוטומטית** לקובץ `data.json` שבתיקייה (בתוך OneDrive),
   ו-OneDrive מסנכרן אותו. למעלה יופיע "מסונכרן עם OneDrive" / "נשמר ל-OneDrive".
4. לסגירה: סגרו את הדפדפן ואת החלון השחור.

> **חשוב:** עבדו תמיד דרך `הפעלה.bat`. אם תפתחו את `index.html` ישירות (כתובת
> שמתחילה ב-`file:///`) — האפליקציה תעבוד, אבל השמירה תהיה בתוך הדפדפן בלבד ללא
> סנכרון אוטומטי ל-OneDrive (אז השתמשו אז ב"הגדרות וגיבוי → הורד/טען גיבוי").

> אם הדפדפן מציג גרסה ישנה — רעננו עם **Ctrl+Shift+R**.
> דרוש דפדפן Edge או Chrome (מותקנים כברירת מחדל ב-Windows).

## המסכים

- **סידור יומי** — בוחרים תאריך, מוסיפים אתרים, משבצים תלמידים, מסמנים נוכחות
  (הגיע / יצא לעבודה). אפשר "לשכפל מיום קודם" או "לטעון מתכנון". כפתורי הדפסה וייצוא אקסל.
- **תכנון שבועי** — לוח שבועי בתאריך עברי לשליחה לצוות (אתר, כמות, קבוצה A/B/C, הערה).
- **דרישת תשלום** — בוחרים חודש, והמערכת מחשבת אוטומטית לכל לקוח את השעות והתשלום
  לפי התעריף השעתי והנסיעות. אפשר לערוך כל שורה. "ייצוא לגבייה" יוצר אקסל מוכן לשליחה.
- **דוחות** — כמה ימים/שעות עבד כל תלמיד, פעילות לפי אתר, וצוות. עם ייצוא אקסל.
- **נתוני בסיס** — תלמידים / אתרים-לקוחות / אנשי צוות / הסעות. הוספה ידנית או **ייבוא מאקסל**.
- **הגדרות וגיבוי** — שם המוסד, שעות ברירת מחדל, וחיבור/גיבוי ל-OneDrive.

## איך מתחילים (פעם ראשונה)
1. **נתוני בסיס → אתרים**: הוסיפו את החקלאים/אתרים עם **תשלום שעתי** ו**תשלום נסיעות** (חשוב לדרישת התשלום).
2. **נתוני בסיס → תלמידים**: הוסיפו ידנית או "ייבוא מאקסל" (אפשר למשוך מקובץ סידור קיים).
3. הוסיפו **אנשי צוות** ו**הסעות**.
4. עברו ל**סידור יומי** והתחילו לשבץ.

## חשוב לדעת
- מומלץ שאדם אחד יערוך בכל פעם. OneDrive מסנכרן קובץ — אם שניים עורכים יחד, השמירה האחרונה גוברת.
- כדאי להוריד גיבוי מדי פעם (הגדרות וגיבוי → הורד גיבוי).
